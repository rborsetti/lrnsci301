HOW TO READ THIS LOGBOOK

DASHED BOX An empty field. You write a number in it (copied from the console).
SOLID BOX A number is already printed for you. Do not change it.
SHADE Colour in this region with your pencil.
⤢ FOLD Fold the paper on this dashed line and crease it.
✂ CUT Cut along this dashed line with scissors.
▲ MARK Make a tick where you decide the split goes.
EVERY SYSTEM, SAME 4 STEPS:
  1. Read the system on the console screen.
  2. Copy its numbers into the dashed boxes on that page.
  3. Work the strip by hand until you can see one unit.
  4. Enter your answer on the console to restore the system.
The console only checks the answer. This logbook is where you actually figure it out. Every system hides the same move: find the value of one unit.
